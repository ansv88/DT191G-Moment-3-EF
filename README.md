# Moment 3 - Boksamling 

## Beskrivning
Denna applikation är en uppgift i kursen DT191G Webbutveckling med .NET som ges via Mittuniversitetet.
Det är en **ASP.NET Core MVC-applikation** som är byggd med **Entity Framework Core**.  
Den hanterar en bokdatabas, där användare kan skapa, läsa, uppdatera och ta bort böcker och författare.

---

## Funktioner
- **CRUD-funktionalitet** för böcker och författare.
- **Databashantering med Entity Framework Core**.
- **Automatiska migreringar** för att skapa och uppdatera databasen.

---

## Teknologier
- **ASP.NET Core MVC**
- **Entity Framework Core**
- **SQLite** (kan bytas till SQL Server eller PostgreSQL)
- **Razor Views**
